# 🛡️ Cyberbullying Support Platform

Fullstack platform developed to help identify, analyze, and respond to cyberbullying situations.

## 🚀 Features
- AI-powered text and image analysis for cyberbullying detection
- Intelligent assistant for guidance and support
- Risk classification system with recommended actions
- Interactive map with nearby police stations (PSP)
- User dashboard with history of analyses and chats
- Educational content about cyberbullying

## 🧠 Tech Stack
Frontend:
- React
- TailwindCSS
- Framer Motion

Backend:
- Node.js
- REST API
- AI-based triage system

Other:
- Google Maps API
- Authentication system
- Responsive design (mobile-first)

## 🎯 Purpose
Designed to provide accessible, structured, and reliable support for victims of cyberbullying, aligned with institutional standards (PSP context).

## 📦 Status
Active development — fully functional core features implemented.
